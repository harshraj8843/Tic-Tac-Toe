## 🧐 About <a name = "about"></a>

A Simple Tic Tac Toe game in Python.

## 🏁 Getting Started <a name = "getting_started"></a>

- clone project
  ```
  git clone https://github.com/harshraj8843/Tic-Tac-Toe
  ```
  ```
  cd Tic-Tac-Toe/
  ```
- run
  ```
  python3 ttt.py
  ```
